# homebrew-tap

Homebrew formulae that allows installation of tools through the [Homebrew](https://brew.sh/) package manager.

## Installation

```
brew install syarihu/tap/<FORMULA>
```

## Formulae

| Repository | Formula | Description |
| ---------- | ------- | ----------- |
| [calday](https://github.com/syarihu/calday) | [formula](Formula/calday.rb) | CLI tool to fetch today's calendar events as JSON |

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh/)
